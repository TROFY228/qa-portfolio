## TC-001: Успешная авторизация

**Заголовок:** Вход с валидными учётными данными

**Предусловия:**
- Пользователь находится на странице https://www.saucedemo.com/
- Поля Username и Password пустые

**Шаги:**
1. В поле "Username" ввести: `standard_user`
2. В поле "Password" ввести: `secret_sauce`
3. Нажать кнопку "LOGIN"

**Ожидаемый результат:**
- Пользователь авторизован
- Открыта главная страница с товарами
- URL: https://www.saucedemo.com/inventory.html
- Отображается приветствие "Welcome"

**Постусловия:** 
- Пользователь может добавить товары в корзину

**Статус:** ✅ Passed

## TC-002: Авторизация с неверным паролем

**Заголовок:** Проверка реакции системы на неверный пароль

**Предусловия:** 
- Пользователь на странице входа

**Шаги:**
1. В поле "Username" ввести: `standard_user`
2. В поле "Password" ввести: `wrong_password`
3. Нажать кнопку "LOGIN"

**Ожидаемый результат:**
- Авторизация не выполнена
- Отображается ошибка: "Epic sadface: Username and password do not match any user in this service"
- Поля очищены
- URL не изменился

**Статус:** ✅ Passed

---

## TC-003: Вход с пустыми полями

**Заголовок:** Проверка валидации пустых полей

**Предусловия:** 
- Страница входа открыта

**Шаги:**
1. Не заполнять поля Username и Password
2. Нажать кнопку "LOGIN"

**Ожидаемый результат:**
- Отображается ошибка: "Epic sadface: Username is required"
- Поля остаются пустыми
- Переход на другую страницу не выполнен

**Статус:** ✅ Passed

## TC-004: Выход из системы (Logout)

**Заголовок:** Проверка функции выхода

**Предусловия:** 
- Пользователь авторизован

**Шаги:**
1. Нажать на меню (три горизонтальные полосы)
2. Выбрать пункт "Logout"

**Ожидаемый результат:**
- Пользователь разлогинен
- Открыта страница входа
- URL: https://www.saucedemo.com/
- Сессия очищена

**Статус:** ⏸️ Not tested
