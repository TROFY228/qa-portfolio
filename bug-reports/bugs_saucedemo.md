# Баг-репорт #001

**Заголовок:** [Web] Сортировка товаров по цене не применяется

**Шаги:**
1. Открыть https://www.saucedemo.com/
2. Авторизоваться: standard_user / secret_sauce
3. Открыть фильтр сортировки (верхний правый угол)
4. Выбрать «Price (high to low)»

**Ожидаемый результат:** Товары отображаются по убыванию цены

**Фактический результат:** Порядок товаров не меняется

**Серьёзность:** S3 (Значительный)  
**Приоритет:** P2 (Средний)  
**Окружение:** Chrome 121, Windows 11, 1920x1080
